##Vim配置笔记  

####从我的github上下载vimrc
```c
git clone https://github.com/shengfzy/Vim-config
```

####安装Vundle插件管理器
```c
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

####安装插件
1. 运行Vim
2. :PluginInstall

####Appendix
######配置的快捷键
|功能|快捷键|
|---|---|
|保存文本|<kbd>,</kbd>+w|
|打开关闭NERDTree|F2|
|打开关闭Taglist|F3|
|打开markdown预览|F8|
|关闭markdown预览|F9|
